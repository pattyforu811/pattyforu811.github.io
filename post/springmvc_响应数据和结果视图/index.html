返回 String

可以用 Model 存储数据

@RequestMapping("/testString")
public String testString(Model model) {
	// 模拟从数据库中查询的数据
	User user = new User();
	user.setUsername("张三");
	user.setPassword("123");
	user.setAge(12);

	model.addAttribute("user", user);
	return "success";
}

 /**
 * return void
 * 可以使用请求转发或者重定向跳转到指定的页面
 */
@RequestMapping("/testReturnVoid")
public void testReturnVoid(HttpServletRequest request, HttpServletResponse response) throws
		ServletException, IOException {
//  1、使用 request 转向页面
//  WEB-INF 前面的/ 不能少, 少了返回的路径会加上RequestMapping的一级目录
//  用request转发不会用到视图解析器, 要把路径写全
//  request.getRequestDispatcher("/WEB-INF/pages/redirect.jsp").forward(request, response);

//  2、也可以通过 response 页面重定向
//  response.sendRedirect(request.getContextPath()+"/redirect.jsp");

//  3、也可以通过 response 指定响应结果
	response.setContentType("text/html;charset=UTF-8");
	response.getWriter().write("卓镇雄");
	return;
}

/**
 * 返回ModelAndView对象
 */
@RequestMapping("/testModelAndView")
public ModelAndView testModelAndView() {
	ModelAndView view = new ModelAndView();
	// 跳转到success.jsp页面
	view.setViewName("success");

	// 模拟从数据库中查询的用户信息
	User user = new User();
	user.setUsername("张三");
	user.setPassword("123");

	//把use对象存储到mv对象中，也会把user对象存入到 request对象
	view.addObject(user);

	return view;
}

SpringMVC框架提供的转发和重定向

forward请求转发

controller 方法在提供了 String 类型的返回值之后，默认就是请求转发。想进行请求转发也可以写成

1
2
3
4

/**
 * 使用forward关键字进行请求转发
 * forward:转发的JSP路径"，不走视图解析器了，所以需要编写完整的路径
 */
@RequestMapping("testForward")
public String testForward() {
  return "forward:/WEB-INF/pages/success.jsp";
}

redirect重定向

重定向 return 就使用 redirect 关键字 ,这里的路径可以不用写项目名, springmvc 会自动帮我们加上。

1
2
3
4

@RequestMapping("testRedirect")
public String testRedirect() {
  return "redirect:/redirect.jsp";
}

@ResponseBody响应json数据

jsp 向服务器发送 json 字符串, 服务器用@RequestBody注解封装到 bean 对象中 ( 把 json 字符串封装到对象要用到 jackson 的jar包 ), 修改 Bean 对象后, 可以直接 return 该对象, 并使用@ResponseBody注解把 bean 对象转为 json 字符串

        $("#responseBody").click(function () {
            $.ajax({
                url: "user/testAjax",
                contentType: "application/json;charset=UTF-8",//发送信息至服务器时内容编码类型
                data:'{"username":"name","password":"123456", "age": 20}',
                datatype:"json", // 预期服务器返回的数据类型
                type:"POST",
                success: function (data) {
                    //data 服务器响应的json数据
                    alert(data.username);
                    alert(data.password);
                    alert(data.age);
                }
            });
        });
    });
</script>
